1. Install required packages using :
	1.Open terminal using "window + R"
		> pip install opencv-contrib-python
        	> pip install cvzone

2. Do the required connections as given in the image .

3. Upload  ".ino" Arduino file to Arduino board 

4. Run the Python(py) file.
